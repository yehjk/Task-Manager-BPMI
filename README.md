# Task Manager System (Kanban) — Student MVP

## Local run

### Client (React)
- folder: `client`
- env: copy `.env.sample` → `.env`
- start: `npm start` (port 3000)

### Server (Express)
- folder: `server`
- env: copy `.env.sample` → `.env`
- start: `npm run dev` (port 3001)

### URLs
- Client: http://localhost:3000
- Server: http://localhost:3001
- Health: http://localhost:3001/health

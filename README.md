# Task Manager System (Kanban) — Student MVP

## Local run

Before using, make sure you have git (not github desktop!!) and node js installed.

vscode:
To use this repo, you will need to press ctrl shift p, select git: clone, paste the link, and log into your Github account. You will then gain access. Don't forget to change the branch! (from main to the one I gave you)

### Client (React)

- folder: `client`
- env: copy `.env.sample` → `.env`
- start: `npm start` (port 3000)

### Server (Express)

- folder: `server`
- env: copy `.env.sample` → `.env`
- start: `npm run dev` (port 3001)

### URLs

- Client: http://localhost:3000
- Server: http://localhost:3001
- Health: http://localhost:3001/health
